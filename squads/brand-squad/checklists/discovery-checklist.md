# Discovery Phase Checklist

## Initial Contact
- [ ] Client contact information obtained
- [ ] Initial kickoff meeting scheduled
- [ ] Team availability confirmed
- [ ] NDA signed (if required)

## Briefing Collection
- [ ] Company background documented
- [ ] Products/services overview obtained
- [ ] Target audiences defined
- [ ] Market landscape understood
- [ ] Competitive analysis requested
- [ ] Business objectives clarified

## Pre-Meeting Preparation
- [ ] Briefing template prepared
- [ ] Discovery guide created
- [ ] Questions compiled
- [ ] Materials reviewed

## Discovery Meeting
- [ ] All stakeholders present
- [ ] Meeting recorded (if approved)
- [ ] Notes documented in real-time
- [ ] Photos/references collected

## Post-Meeting
- [ ] Briefing document completed
- [ ] Client feedback incorporated
- [ ] Team review meeting held
- [ ] Gaps identified and addressed
- [ ] Final briefing approved

## Readiness for Next Phase
- [ ] Briefing complete and approved
- [ ] All team members briefed
- [ ] Competitive analysis assigned
- [ ] Strategy phase scheduled
