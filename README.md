# buildado

Synkra AIOS Project - Full Stack Development Framework

## Getting Started

### Prerequisites
- Node.js 18+
- Git
- GitHub CLI

### Setup

```bash
npm install
```

### Development

```bash
npm run dev
```

## Project Structure

```
docs/           # Documentation
├── stories/    # Development stories
├── prd/        # Product requirement documents
└── architecture/ # Architecture documentation

packages/       # Project packages
squads/         # Team/squad modules
tests/          # Test files
```

## AIOS Integration

This project uses Synkra AIOS for orchestrated AI-driven development.

See `.aios-core/` for framework details.
